📋 How to use this app
Step 1
Enter NDC-11
Scan or type the 11-digit NDC from the purchasing (outer) UOM. The app will check if this item has been scanned before and look up its characteristics from the master database.
Step 2
Confirm Characteristics
Review the UOM1, UOM2, and UOM3 values pulled from the master. Click ✓ Correct if they match the item in hand. If wrong, click ✎ Incorrect — Edit to enter the right values. If you can't tell, click ⚠ Unable to Confirm.
Step 3
Scan Barcodes
Select Purchasing for the outer/case barcode or UOM1 (if different) for the inner/each barcode. Scan with the connected reader or type manually, then press Enter. Scan all barcodes present — if an item has both a UPC and a DataMatrix (or any other combination), scan each one separately before moving on. Add a note if needed, then click → Next Item when done.
💡 Tip: If the NDC on the purchasing UOM and the inner UOM (UOM1) are the same, you can scan all barcodes under Purchasing — there is no need to use the UOM1 button in that case.
💊 NDC-11 REQUIRED
This should be the NDC on the purchasing UOM without any dashes. 11-digit NDC format is 5-4-2 and a 0 should be added at the front of the segment that is missing a number (example: 12345-678-99 would become 12345-0678-99 and entered as 12345067899 in the field).
